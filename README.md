# CS3SP-MovieList

## Development
1. git clone https://github.com/yelloweq/CS3SP-MovieList.git
2. cd CS3SP-Movielist
3. npm install
4. composer install
5. enter details in .env file
6. create and seed db:
     php db_setup.php
7. configure apache document root to path/to/dir/public
8. run apache server
